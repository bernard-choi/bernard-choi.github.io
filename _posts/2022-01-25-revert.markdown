## 테스트를 위한 파일 생성
테스트를 위해 아래와 같이 파일을 생성함

```python
## a.py
a 파일을 생성함
```

```python
## b.py
b 파일을 생성함
```

```python
## c.py
c 파일을 생성함
```
---

## git revert로 파일 변경사항 초기화히기

- b 파일에 문제가 있어 이를 삭제하려고 하는데 `reset`을 쓰면 그 이후 커밋인 c파일도 사라짐
- 이 경우 revert를 사용하여 b파일을 생성한 커밋만 삭제할 수 있음
  - `git revert {커밋 해시}`
  - `a.py`와 `c.py`만 남아있음
- `revert`의 경우 `reset`과는 다르게 로그 기록이 남는다.
